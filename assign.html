Assignments and Projects

Assignment 1 : Report

Students must choose a group of websites from the same domain or sector and create performance statistics for each using Google Lighthouse. These reports need to be analyzed to determine their strengths and shortcomings, the results need to be compared to show any noteworthy changes or trends, and the entire review process needs to be documented in a detailed report. Recommendations for optimization and enhancement of every website should be included in the report. The websites that are chosen ought to have high search engine rankings and come from reliable sites like Fortune, Forbes, Statista, QS Ranking, and Alexa.

Assignment 2: Website

Using HTML and CSS Grid layout, create a landing page for your personal website by designing it based on one of the pre-made themes. Divide the page into a header, content section, footer, and left and right menus. A welcoming heading and a brand image should be included in the header. Navigational links should be included in both menus, and the content area should provide interesting writing and graphics that showcase accomplishments, hobbies, and personal information. There should be a copyright notice in the footer. Use CSS styling to make the design more aesthetically pleasing, make sure it is responsive, and make it uniquely yours. To explain the rationale behind each section's purpose and design choices, provide internal comments in the HTML and CSS code.

Assignment 3: GitHub E-portfolio

Create a personal GitHub e-portfolio for each student that includes a thoughtful selection of their work that demonstrates their learning objectives, accomplishments, and talents. Profile parts containing personal data, education, jobs, talents, courses, certificates, honors, awards, and projects should be included in the e-portfolio. Together with a project, it should also include assignments like a website, report, and the e-portfolio itself. Every assignment and project should have a reflection that summarizes the lessons learned, makes suggestions for improvement, and adds any additional creative ideas or personal observations.
